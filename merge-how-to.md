## Получение ветки с сервера

Для получения ветки с сервера используйте команду `git fetch`, за которой следует имя удаленного репозитория (обычно `origin`), и имя ветки:

```bash
git fetch origin branch_name
Затем вы можете переключиться на эту ветку с помощью команды git checkout:

bash
Copy code
git checkout branch_name
Слияние одной ветки с другой
Сначала переключитесь на ветку, с которой вы хотите выполнить слияние (например, master или main):

bash
Copy code
git checkout master
Затем выполните команду git merge, за которой следует имя ветки, которую вы хотите слить:

bash
Copy code
git merge branch_name
Что делать после слияния веток
После слияния веток, если все прошло успешно и нет конфликтов, вы можете отправить изменения на сервер с помощью команды git push:

bash
Copy code
git push origin master
Если после слияния возникли конфликты, вам нужно их решить. Откройте файлы с конфликтами, найдите места, отмеченные как <<<<<<<, ======= и >>>>>>>, и решите, какие изменения вы хотите оставить. Затем добавьте исправленные файлы с помощью git add и сделайте коммит с помощью git commit.
